## OWL UI Components
OWL UI is a Vue 2.0 **Component Design System**, it works in Cepave's monitoring system OWL.

## Install

```sh
$ npm i --save github:cepave-f2e/vue-owl-ui
```

## Usage

```js
import {Grid, Select, Tip} from 'vue-owl-ui'
```
